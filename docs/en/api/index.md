# API Reference

::: warning Notice
This page is under construction, content may change.
:::

LinguaFlow provides a RESTful API for translation management. Detailed API documentation will be provided in a future release.

## Base Information

- Base URL: `http://localhost:8080/api/v1`
- Authentication: Bearer Token

## Main Endpoints

- `GET /translations` - List translations
- `POST /translations` - Create translation
- `GET /translations/:id` - Get translation by ID
- `PUT /translations/:id` - Update translation
- `DELETE /translations/:id` - Delete translation
