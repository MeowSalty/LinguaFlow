## Environment Variables

LinguaFlow supports the following environment variables for configuration:

| Variable | Description | Default |
|----------|-------------|---------|
| `LINGUAFLOW_PORT` | Server listening port | `8080` |
| `LINGUAFLOW_DB_PATH` | Database file path | `./data/linguaflow.db` |
